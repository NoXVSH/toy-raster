
这是一个使用typescript的光栅化渲染程序，目前支持的功能有

  * 画线/三角形
  * 透视贴图
  * 视锥裁剪
  * 边缘剔除
  * 深度缓冲
  * 纹理双线性插值
  * 2x2的MSAA

计划中：

  * 模型显示
  * 基本光照
  



##### 一. 运行

 使用浏览器打开out/demo.html


##### 二. 修改ts代码后如何编译

执行build即可

不过需要配置好编译环境（tsc+webpack)

执行

npm install --only-dev 

即可
